# Git Commands Reference

## Setup & Config

### git --version
Shows installed Git version
Example:
git --version

### git config --global user.name
Sets global username
Example:
git config --global user.name "John Doe"

### git config --global user.email
Sets global email
Example:
git config --global user.email "john@email.com"

---

## Basic Workflow

### git init
Initializes a new Git repository
Example:
git init

### git status
Shows current repository state
Example:
git status

### git add
Stages changes for commit
Example:
git add git-commands.md

### git commit
Creates a commit from staged changes
Example:
git commit -m "Add initial Git commands reference"

---

## Viewing Changes

### git log
Shows commit history
Example:
git log

### git log --oneline
Shows compact commit history
Example:
git log --oneline

### git diff
Shows unstaged changes
Example:
git diff
